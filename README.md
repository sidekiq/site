# sidekiq.org

This is the source code for [https://sidekiq.org](https://sidekiq.org).
We use Bridgetown as our static site generator and pull in the latest wiki content from Git every time the site is deployed.

## Gettting Started

```
bundle install
```

## Development

Fun commands:

```
bin/bt deploy       # build site into output/
bin/bt s            # run a localhost:4000 server
open http://localhost:4000
```
